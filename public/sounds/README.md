# Chat Notification Sounds

Place notification sound files in this directory.

## Required Files

- `notification.mp3` - Played when a new message is received

## Recommended Specifications

- Format: MP3
- Duration: 0.5 - 2 seconds
- Sample Rate: 44100 Hz
- Bit Rate: 128 kbps or higher

## Free Sound Resources

You can download free notification sounds from:
- [Mixkit](https://mixkit.co/free-sound-effects/notification/)
- [Freesound](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)

Rename the downloaded file to `notification.mp3` and place it in this folder.
