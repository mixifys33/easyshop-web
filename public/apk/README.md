# APK Downloads Folder

Drop your `.apk` files here and they will automatically appear on the `/install` page.

## Naming convention (recommended)

`EasyShop-v1.0.0.apk`, `EasyShop-v1.1.0-preview.apk`, etc.

The install page reads this folder via the `/api/apk-releases` API route and displays each file with its name, size, and a direct download link.
